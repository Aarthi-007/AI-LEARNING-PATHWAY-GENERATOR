
# AI Learning Pathway Generator 🚀

An AI-powered web app that generates **personalized learning roadmaps** based on a user's goals, interests, and skill level. It helps learners discover structured steps, recommended technologies, and resources to achieve their learning objectives.

---

## ✨ Features

* 🤖 AI-generated learning pathways
* 🎯 Personalized based on user goals
* 📚 Recommended technologies & resources
* ⚡ Modern UI with React
* 🔗 Backend API integration
* 📱 Responsive design

---

## 🛠 Tech Stack

**Frontend**

* React
* TypeScript
* Vite

**Backend**

* Node.js
* Express
* TypeScript

**Tools**

* Git & GitHub
* Environment Variables (.env)

---

## ⚙️ Installation

```bash
git clone https://github.com/HarishPatil20/AI-Learning-Pathway-Generator.git
cd AI-Learning-Pathway-Generator
npm install
npm run dev
```

---

## 🚀 Usage

1. Enter a learning goal (e.g., Web Development, AI).
2. The AI generates a structured learning roadmap.
3. Follow the recommended steps and technologies.
